We predominantly use Psychtoolbox (version 3.0.13) to run psychophysics and/or eye tracking experiments. As per PTB recommendation, the lab PC runs Linux (Linux Mint 18 Sarah). Notes for setting up this system and getting it to work with PTB, ViewPixx, Eyelink and PsychoPy are documented here.

Starting a PTB experiment

 

  • The Eyelink operator PC and the subject PC are stacked on top of each other. The top box is the subject PC; the bottom one is the operator PC.
  • On the subject PC, log on as 'eyelink'. You will have been given the password.
  • Start a command terminal. Either press [Ctrl+Alt+T] or go to Applications (top left corner) - Accessories - Terminal.
  • Start matlab by typing: 'matlab'. 
  • In Matlab, navigate to the folder than contains your experimental files. Typically, your experimental folder will be stored in 'Documents/MATLAB/PTB_experiments'. You can navigate this location using the GUI interface or by typing 'cd Documents/MATLAB/PTB_experiments/<yourfoldername>' in the Matlab command window.
  • Start your experiment by typing in the relevant command in the Matlab command window (e.g. 'Experiment4(<subject nr>,<block nr>);').
  • Check how to operate the eye tracker.
  • PTB will initially show a white screen during which it is doing a number of tests, such as measuring the interval between different frames. When these tests finish, the screen will change to whatever it was programmed to be (e.g. a blank mid-grey screen). At the end of your experiment the Matlab window will re-appear, but in an eye tracking experiment the data will at that point be transferred from the operator PC to the display PC. So watch the lower left corner of your Matlab window to see if it is ‘busy’ or not.

 

Both PTB and Ubuntu are very actively maintained and regularly updated. Certainly as far as PTB is concerned it is a good idea to stay up-to-date.

 

If you think an update is in order - of either PTB or the OS - check with CL and other lab members whether and when the update should be scheduled for!

 

However, bear in mind that some of the functions may have been changed by lab members, in which case an update might replace the altered files. A list of functions that have been edited (without renaming) is given below. Copies of these are stored in root/Documents/MATLAB/Psychtoolbox). After updating PTB, you can copy these functions into the relevant PTB folder(s) (provided nothing in these particular functions has changed too drastically in the newer PTB release). Make sure to alter the 'Last updates' line below when you have made an update, so that other lab members know what has happened. Disconnect the PC from the network once you have completed the relevant updates.

 

Whenever you edit a function that is likely to be used in experiments run by other lab members, change its name so you can call the function you specifically need. When you edit a function without renaming, please add it to the following list.
